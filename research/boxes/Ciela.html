Determing Gravitational Lensing Parameters Using Convolutional Nueral Networks

Edgar P. Vidal, Sanskruti Admane, Sophia Da Costa

Astromatic Hackathon Presentation and Code!

We worked in a team to create a simulation based machine learning model that can infer gravitational lensing parameters (30h hackathon!). Typically, this determination takes on average 1-2 days per lens, but by training on a simulator, we get aceess to an infinite data set and can determine the parameters of thousands of lensed images in seconds. Our project and presentation was awarded first place by a panel of 3 leading experts in astronomy at the Ciela Institute. If you are interested, Convolutional Nueral Network model is available on my github linked below! The simulator, designed by Alexandre Adam (Ph.D candidate) remains private however.
